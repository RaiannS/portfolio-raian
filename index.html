Analyse de Trames Réseau
Wireshark · Environnement Virtualisé · VMware
Environnement technique
Machine cliente
POSTE-1 — génération du trafic
Machine serveur
POSTE-2 — FTP, messagerie, web
Machine sniffeur
POSTE-3 — Wireshark
Virtualisation
VMware
Réalisation
  • Installation et configuration d'un serveur FTP avec FileZilla Server — création d'un compte utilisateur Lucky
  • Déploiement d'un serveur de messagerie avec hMailServer — comptes créés dont lucky@farwest.bd
  • Mise en place d'un serveur web avec XAMPP et création d'un site HTML de test
  • Génération de trafic réseau : ping (ICMP), requêtes DNS, navigation web, renouvellement DHCP (Discover → Offer → Request → ACK)
  • Envoi et réception de courriels avec Thunderbird — analyse SMTP, POP, IMAP
  • Capture et analyse des trames en temps réel depuis la machine sniffeur
Résultats observés
  • Trames ARP : résolution adresse IP ↔ adresse MAC
  • Identifiants FTP visibles en clair — démonstration des risques des protocoles non sécurisés
  • Données de formulaires HTTP lisibles vs HTTPS chiffré et illisible
  • Rôle des ports réseau : port 21 (FTP), port 53 (DNS) — ports fixes serveur vs ports éphémères client
▶ Ce TP m'a permis de mieux comprendre comment les machines communiquent sur un réseau. J'ai appris à utiliser Wireshark pour analyser les échanges et à identifier les informations visibles ou protégées selon les protocoles utilisés.

Déploiement Automatisé de Postes
Stage CCI Lyon · Dell Latitude · PXE · Image Système
Contexte & Besoin
  • Stage au service informatique de la CCI Lyon Métropole Saint-Étienne Roanne
  • Plusieurs dizaines de postes à configurer avec le même environnement logiciel
  • Automatisation pour limiter les interventions manuelles et garantir une configuration homogène
Environnement technique
Matériel
Dell Latitude 3500 / 3520 / 3540
Démarrage
PXE (réseau)
Méthode
Image système (master)
Infrastructure
Serveurs CCI
Réalisation
  • Préparation du poste maître : installation OS, pilotes et logiciels requis par les formations
  • Renommage du poste selon la nomenclature définie par le service informatique
  • Capture complète du système pour créer l'image de référence (master)
  • Enregistrement de l'image sur le serveur de déploiement
  • Déploiement automatisé via le réseau (PXE) sur plusieurs dizaines de machines
  • Finalisation post-restauration : renommage, vérification logiciels, suppression profils temporaires, contrôle de conformité
  • Tests complets : démarrage, applications, fonctionnalités attendues
▶ Cette mission m'a permis de développer mes compétences en déploiement automatisé, en gestion d'images système et en administration de parc. J'ai compris l'intérêt de standardiser les configurations : cela facilite la maintenance, réduit le temps de préparation et permet d'obtenir un parc informatique homogène. J'ai également gagné en autonomie en appliquant des méthodes professionnelles réelles.
